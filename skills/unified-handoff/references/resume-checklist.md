# Resume Checklist

Before editing:

- [ ] Read the latest validated handoff completely.
- [ ] Confirm project root and working directory.
- [ ] Confirm branch and HEAD commit.
- [ ] Run the staleness check.
- [ ] Review commits and changed files since the recorded HEAD.
- [ ] Confirm critical referenced files still exist.
- [ ] Re-check blockers and open questions.
- [ ] Validate every Unverified Assumption before relying on it.
- [ ] Treat explicit User Corrections as higher priority than old agent prose.
- [ ] Do not repeat a failed attempt marked `Do Not Repeat: Yes` unless its retry condition is satisfied.
- [ ] Start with Immediate Next Steps item 1.

When pausing again, create a new draft with `continues_from` pointing to the current archive rather than modifying an old validated file.
